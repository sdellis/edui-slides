Hands on Section

Search for a GIF

Criteria:

  • Biological in nature
  • Subject should stay completely within frame at all times
  • Well lit subject
  • At least 400px on smallest side
  • Less than 30 frames (hard to tell, so gather a few possibilities)

Start tracing joints

Two ways to use the interface:

  1. On the web: Choose a GIF
  2. Locally: clone my repository... and add your own gif to one of the example files. If you have npm/gulp installed, follow these steps:
$ git clone https://github.com/sdellis/svg-draw-component.git
$ cd svg-draw-component
$ npm install
$ gulp
$ gulp examples

Steps

  1. “Trace” your first frame
  2. Take a snapshot
  3. “Trace” your next frame… and so on
  4. Export Code

Paste into your CodePen template

Tag it as kinetic-constellation, and add it to your showcase.
